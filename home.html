DojoGram A Single-Page Social Media App

With social media encompassing a huge part in everyone's lives, I decided to demonstrate my ability to form a simple, fluid social media app. I believe the MEAN Stack handles social media best with its single-page aesthetic giving the user zero load time when navigating through the app. This app uses Passport.js for user authentication and Bootstrap's mobile-friendly scaling allows easy access to this site.

I built this from the ground up with the no-SQL MongoDB database, self-joined user association allows users to follow or unfollow other users and populates posts accordingly.

GifParty A Media Player and Visualizer

If you're ever hosting a party and need something both visually and aurally fun to play in the background, check out GifParty! Here, I used both Giphy and Soundcloud's API to let users choose what genres they want to listen to, and what gif's they want to see play in the background! GifParty is built with Python and the Pylot MVC as its framework to login and register users, although not much more functionality is built with it yet, as I develop this webApp I plan to store users' searches and be able to play back certain playlists/gif searches.

Unfortunately, GifParty is not live at the moment; however, if you'd like to check out the source code you can find it here.
